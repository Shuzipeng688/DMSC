# DMSC - 冒泡排序算法实现

本项目实现了经典的冒泡排序算法，包含完整的功能演示和逐步排序过程展示。

## 项目简介

冒泡排序是一种简单的排序算法，通过重复地遍历要排序的列表，比较相邻的元素并交换它们的位置，直到整个列表有序。

## 功能特性

- 标准冒泡排序实现
- 优化版本（提前终止）
- 多种测试用例演示
- 逐步排序过程展示（适合教学）

## 算法特性

- **时间复杂度**: O(n²)
- **空间复杂度**: O(1)
- **稳定性**: 稳定

## 使用方法

### 运行演示程序

```bash
python bubble_sort.py
```

### 使用排序函数

```python
from bubble_sort import bubble_sort

# 对数组进行排序
arr = [64, 34, 25, 12, 22, 11, 90]
sorted_arr = bubble_sort(arr)
```

```python

bubble_sort_step_by_step(arr)
```

## 算法原理

冒泡排序的基本思想：

1. 比较相邻的两个元素，如果前一个元素大于后一个元素，则交换它们
2. 对每一对相邻元素重复上述操作，从开始的第一对到结尾的最后一对
3. 完成一轮后，最大的元素会"冒泡"到数组末尾
4. 对剩余元素重复以上步骤，直到排序完成

## 优化说明

本实现包含以下优化：

- **提前终止**: 如果某一轮没有发生交换，说明数组已经有序，可以提前结束排序
- **减少比较次数**: 每一轮后，最大的元素已经位于正确位置，可以减少后续比较次数
